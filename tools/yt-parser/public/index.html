Нет ошибок
Admin API Token
Если в .env задан ADMIN_TOKEN, все мутации API (старт/стоп агента, approve сделок, правка лидов и т.д.) требуют этот токен в заголовке x-admin-token. Введи тот же токен здесь — он сохранится в localStorage и будет подставляться автоматически.
Follow-up (напоминания молчащим)
Если блогер не ответил через N дней после последнего сообщения агента — автоматически отправить вежливое напоминание. Максимум M попыток на одного лида, потом оставляем в покое.
Review Mode (проверка ответов)
Если включено — AI-агент не отправляет сообщения блогерам напрямую, а складывает их в очередь во вкладке «На проверке». Ты просматриваешь, редактируешь и одобряешь/отклоняешь каждый ответ. Рекомендуется включить на время обкатки агента.
Переключатель применяется мгновенно — перезапускать сервер не нужно.
Telegram Outreach (gramjs)
Залогинь обычный Telegram-аккаунт через MTProto, чтобы AI-агент мог писать блогерам в личку. ⚠️ Используй отдельный аккаунт — массовая рассылка может привести к бану.
Шаги:
1. Получи API_ID и API_HASH на my.telegram.org/apps
2. Пропиши в .env: TG_API_ID, TG_API_HASH, TG_PHONE
3. Перезапусти сервер
4. Нажми «Login» — придёт SMS с кодом
5. Введи код, при необходимости 2FA-пароль
Email Setup (Resend)
Настройки email индивидуальны для каждого проекта. Отправка через Resend.com, приём через IMAP (опционально).
IMAP (для приёма ответов, опционально)
YouTube API Key Pool
Добавляй несколько Google API ключей — парсер будет автоматически ротировать их при исчерпании квоты. Каждый ключ = 10,000 units/день. Ключи можно привязать к конкретному workspace или оставить в общем пуле.
📦 Массовый импорт (вставь несколько ключей)
